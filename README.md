# 🧮 Simple Python Calculator

A lightweight, interactive command-line calculator built using Python. It supports basic arithmetic operations, includes input loops, and features custom Roman-Urdu messaging for user prompts and error handling.

---

## 🚀 Features

* **Core Math Operations**: Instantly calculates addition, subtraction, multiplication, and division.
* **Smart Error Handling**: Built-in guard preventing crashes when dividing by zero (`0 sy divide nhi hoga`).
* **Interactive Loop**: Keeps running continuously so you can perform multiple calculations.
* **Quick Exit**: Simply type `exit` during the operator prompt to close the program safely.

## 🛠️ How to Run

### Prerequisites
Make sure you have [Python](https://python.org) installed on your machine.

### Execution Steps
1. Clone this repository or download the source file.
2. Open your terminal, command prompt, or PowerShell in the project directory.
3. Run the application using:
   ```bash
   python calculator.py
   ```

---

## 💻 Code Structure

The program splits logic into standalone, reusable functions before executing the runtime loop:

* `plus(a, b)`: Calculates the sum of two floats.
* `minus(a, b)`: Calculates the difference between two floats.
* `multiply(a, b)`: Outputs the product of multiplication.
* `divide(a, b)`: Validates denominator safety before returning the quotient.
